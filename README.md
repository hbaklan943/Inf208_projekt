# Inf208_projekt

This is the repository for the Inf208 project at the Turkish German University, where we are developing a security system using Raspberry Pi and different sensors.
# Project Structure
The project is structured as follows:

```
├── README.md
├── guvenlik.py
```

Used sensors:
- PIR Sensor
- Ultrasonic Sensor
- Temperature Sensor

Used interfaces:
- GPIO
- I2C
- Buzzer
- LED
- Servo Motor
